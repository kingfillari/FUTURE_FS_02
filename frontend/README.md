# Mini CRM – Frontend (Task 2)

This is the React frontend for the Client Lead Management System.  
It communicates with the Node.js/Express backend and provides:

- Secure admin login (JWT)
- Lead listing with filtering, sorting, pagination
- Lead status update (New → Contacted → Converted)
- Add/delete notes per lead with optional follow‑up date
- Responsive design
